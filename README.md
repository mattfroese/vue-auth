# vue-auth

`vue-auth` makes managing authenticate state easy in with Vue.

## Installation

### NPM

```
$ npm install @saltyquark/vue-auth
```

## Example

```javascript
import VueAuth from 'vue-auth'
Vue.use(Auth, {
    url: "/auth", // full url to your auth endpoint
})
```
